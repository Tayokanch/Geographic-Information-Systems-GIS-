## WHAT IS GIS

- `GIS`: Geographical Information System is a software tools that can work with spatial data.

- `GIS`: Is used for *capturing* , *representing*, *processing*, and *analyzing* geographic information

# GEOSPATIAL Information Science

- This refer to the entire descipline / field of Geospatial science and the specific tools we use in the descipline is `Geospatial Information System`


## WHAT CAN GIS SOFTWARE DO

`QGIS`: Its one of a GIS Software 
  - It can be used for Viewing of Spatial Data or a Data in a Spatial Format
  - Select features and Zooming into each region
  - Create Spatial Data
  - Create Maps
  - Analyze Spatial Data
  - ETL (Extract - Transform - Load) Spatial Data
