
`data source manager` : This is one tools of the toolbar in QGIS which  your central hub for **adding data** to our project.

**WHAT IS IT USED FOR**

- it Lets you *browse*, *select*, and *load data* from various sources into your QGIS map project.
  
- Supports multiple data formats, including:
  - Vector layers (points, lines, polygons) – e.g., Shapefiles, GeoJSON, GPKG

  - Raster layers (images, elevation, satellite) – e.g., GeoTIFF, JPEG

  - Delimited text (CSV, TXT) with coordinates

  - Database connections – e.g., PostGIS, SpatiaLite
  Web services – e.g., WMS, WFS, XYZ tiles, QuickMapServices`
`

# There different types of Data Source Managers

- `Delimited Text`: A delimited text file is a plain text file where columns (fields) are separated by a specific character, called a delimiter.
  - Common delimiters include: **Comma (,) → CSV files Tab (\t) → TSV files Semicolon (;) Pipe (|)**
