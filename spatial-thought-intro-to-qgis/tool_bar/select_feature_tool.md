## SELECT BY VALUE IN QGIS

- This is used for **querying** a set of data based on some attributes in the **atrribute table**, and after writting the atrribute we click on `select feature` to determine the result of our attribute on the map


## HOW TO DISPLAY THE SELECT VALUE OR ATTRIBUTE ON THE MAP

- Go back to the Layers Panel, right click on the `data set` which we had selected from
- Click on `export` and click `Save Selected features As`
- Choose `location` and a `filename` , then export
