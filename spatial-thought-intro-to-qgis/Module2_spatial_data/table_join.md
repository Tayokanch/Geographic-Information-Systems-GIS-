## TABLE JOIN in QGIS

- To perform a **Table Join** we would use the `Join Attribute by field value` in the ** processing Toolbox**
- `input Layer`: This is usually the data with the **Polygon**
- `Table field`: This is the field with the common value that we wanna use for joining
- - `input Layer2`: The Second Table 
- `Table field2`: The matching field select for first Layer
- Select` Layer field` to copy
- select `type of join`(e.g one-one join)
- select `join layer` click on `save file`, then pick the directory to save to and give a descriptive filename
