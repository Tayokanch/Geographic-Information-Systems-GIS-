# QGIS APPLICATION

- `QGIS DESKTOP`: This is used for **Creating**, **Analysing**, **Editing**, **Vidualizing** and **publishing** Geospatial Informatiom.
- `QGIS SERVER`: This is used for QGIS *projects* and *layer* as OGC compatible WMS and WFS services
- `QGIS WEB CLIENT`: This is use for **publishing** QGIS projects on the web with ease
